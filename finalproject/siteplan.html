YamaWise Site Plan

Site Name

Site Name: YamaWise

Reason: The name YamaWise combines “Yama,” meaning “shine” in the Ibibio language of Akwa Ibom, Nigeria, and “Wise,” reflecting intelligence and smart decisions. Together, the name represents “shining with wisdom,” which fits the site’s mission to help users make better financial decisions.

Optional Domain Availability: yamawise.com

Site Purpose

The purpose of YamaWise is to help individuals and families take control of their finances through AI-powered tools, easy tips, and clear financial education. The website will provide:

  • AI-based budgeting and expense tracking tools
  • Daily money-saving tips and reminders
  • Financial literacy guides and tutorials
  • Resources to reduce debt, plan savings, and improve credit

Scenarios

  • “I’m living paycheck to paycheck. How can I start budgeting without feeling overwhelmed?”
  • “We want to save for a family vacation, but we don’t know where to begin. How can we track our spending better?”

Color Schema

  • Primary Color: Teal (#009688)
  • Secondary Color: Dark Teal (#004d40)
  • Background: Light Gray (#f9f9f9)
  • Text Color: Dark Gray (#333)

Typography

  • Headings: Playfair Display — Used for titles and headings
  • Body Text: Roboto — Used for all standard text
